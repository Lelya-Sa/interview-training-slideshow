# LeetCode - Interview Material

## Definition
LeetCode is a platform for practicing coding interview questions. This section contains common algorithmic problems and their solutions.

## Problem Categories

### Arrays & Strings
- Two Sum
- Maximum Subarray
- Longest Substring
- Group Anagrams

### Linked Lists
- Reverse Linked List
- Merge Two Sorted Lists
- Detect Cycle

### Trees
- Maximum Depth
- Same Tree
- Invert Binary Tree

### Dynamic Programming
- Climbing Stairs
- House Robber
- Longest Common Subsequence

### Graphs
- DFS/BFS
- Shortest Path
- Topological Sort

## Study Approach

1. Understand the problem
2. Identify patterns
3. Plan solution
4. Code solution
5. Test edge cases
6. Optimize time/space complexity
