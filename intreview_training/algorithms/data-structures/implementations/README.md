# Data Structures Implementations

This directory contains complete implementations of all major data structures with all their operations in **three programming languages**: JavaScript, Python, and Java.

## Language Folders

- **js/** - JavaScript implementations (Node.js compatible)
- **python/** - Python 3 implementations
- **java/** - Java implementations

## Available Data Structures

Each language folder contains implementations of:

1. **Array** - Dynamic array implementation
2. **Linked List** - Singly linked list
3. **Stack** - LIFO data structure
4. **Queue** - FIFO data structure (includes Priority Queue, Circular Queue, Deque)
5. **Hash Table** - Key-value storage with collision handling
6. **Binary Search Tree** - BST with all traversals
7. **Heap** - Min and Max heap implementations
8. **Graph** - Graph with DFS and BFS traversals
9. **Trie** - Prefix tree for string operations

## Running Examples

### JavaScript
```bash
node js/array.js
node js/linked-list.js
node js/stack.js
```

### Python
```bash
python python/array.py
python python/linked_list.py
python python/stack.py
```

### Java
```bash
javac java/Array.java
java Array
javac java/LinkedList.java
java LinkedList
```

## Learning Path

1. Start with **Array** - simplest structure
2. Move to **Linked List** - understand pointers/references
3. Learn **Stack** and **Queue** - linear structures
4. Study **Hash Table** - key-value storage
5. Master **Binary Search Tree** - hierarchical data
6. Understand **Heap** - priority-based structure
7. Explore **Graph** - complex relationships
8. Learn **Trie** - string-specific structure
